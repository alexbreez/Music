# 🎵 Music Harmony Analyzer

Анализатор треков по методологии **«Современная гармония»**.

## Что умеет

| Функция | Описание |
|---------|----------|
| **Тональность и лад** | Определение ключа через корреляцию Крумхансл-Шмуклер (мажор, минор, миксолидийский, дорийский, лидийский) |
| **Темп (BPM)** | Автоматическое определение через beat tracking |
| **Распознавание аккордов** | Хромаграмма + шаблонная корреляция (13 типов аккордов × 12 корней) |
| **Функциональный анализ** | Ступени в римских цифрах, выявление заимствованных аккордов |
| **Паттерны гармонии** | Рок-каденция (bVI-bVII-I), обманная каденция, полууменьшенные аккорды |
| **Сегментация** | Разбиение на секции (куплет/припев) по энергии и тембру |
| **Мелодические акценты** | Ступени мелодии относительно аккорда и тоники (флаг `--full`) |
| **Спектральный анализ** | Яркость, динамический диапазон |

## Установка

```bash
pip install -r requirements.txt
```

## Использование

```bash
# Базовый анализ
python analyze_music.py track.mp3

# Полный анализ (с мелодическими акцентами)
python analyze_music.py track.mp3 --full

# Вывод в JSON (для интеграции с UI)
python analyze_music.py track.mp3 --full --json
```

## Методология

Основа анализа — документ «Современная гармония», покрывающий:
- Функциональный анализ по ступеням
- Заимствованные аккорды и модальные замены
- Расширения (9, 11, 13)
- Мелодические акценты относительно гармонии
- Динамический контраст секций
- Полууменьшенные аккорды, обманные каденции, рок-каденции, лидийские аккорды
