# 🎵 Music Harmony Analyzer v4

Анализатор треков по методологии **«Современная гармония»**.

## Распознавание аккордов

Три метода, автоматический выбор лучшего:

| Метод | Точность | Требования | Описание |
|-------|----------|------------|----------|
| **CNN + CRF** | Высокая | `madmom` | Свёрточная сеть + Conditional Random Field |
| **Deep Chroma + CRF** | Средняя | `madmom` | Нейросетевая хромаграмма + CRF |
| **DSP** | Базовая | только `numpy`, `scipy` | HPSS + шаблонная корреляция (фоллбэк) |

## Установка

```bash
pip install -r requirements.txt
```

> madmom автоматически скачает предобученные модели при первом импорте.

## Использование

```bash
# Автовыбор метода (CNN → DeepChroma → DSP)
python analyze_music.py track.mp3

# Конкретный метод
python analyze_music.py track.mp3 --method cnn
python analyze_music.py track.mp3 --method deepchroma
python analyze_music.py track.mp3 --method dsp

# JSON для интеграции с UI
python analyze_music.py track.mp3 --json
```

## Что анализирует

- **Тональность и лад** — по каденции основного паттерна + частотность
- **Темп (BPM)** — onset autocorrelation
- **Аккорды** — нейросетевое или DSP-распознавание
- **Функциональный анализ** — ступени (римские цифры), заимствованные аккорды
- **Паттерны** — повторяющиеся прогрессии, основное/дополнительное движение
- **Паттерны «Современной гармонии»** — пикардийская терция, рок-каденция, обманная каденция, полууменьшенные аккорды

## Методология

Документ «Современная гармония» — основа анализа. Покрывает функциональный анализ по ступеням, заимствованные аккорды, модальные замены, расширения (9, 11, 13), мелодические акценты, полууменьшенные аккорды, обманные каденции, рок-каденции.

## История версий

- **v1** — базовый librosa (темп, тональность, яркость)
- **v2** — HPSS + тактовое окно + шаблоны
- **v3** — Harmonic Change Detection + басовая изоляция + паттерн-матчинг
- **v4** — нейросетевое распознавание (madmom CNN/CRF) + DSP-фоллбэк
